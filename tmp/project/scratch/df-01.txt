Application Configuration
This section describes how to define important application-wide settings that are applied to a
Dayforce site. If you are administering more than one site, each site (or client) can have its own
settings.
Note: For your privacy and security, Dayforce terminates your session and logs you out if your
session is idle or inactive for 30 minutes.
Site Setup
In the Site Setup feature in Dayforce, you can view and set up organization-specific information,
client (customer) details, culture settings, and image files for logos used by the organization, on
checks and earning statements, as well as signatures used on checks.
Before You Begin: Some of the fields or features described in this topic might be available only to
users with Support or Service roles. For more information, contact your implementation consultant.
Depending on how your organization is configured, the Site Setup can have the following
subfeatures:
Client: Contains identifying information about the organization, name, namespace, and
reference code. See Site Setup > Client.
Clock Devices: Lists any clocks or clock types associated with the organization. See Clock
Device Setup.
Clock Config Groups: Lists any clock configuration groups associated with the organization.
See Create Clock Configuration Groups.
Clock Configuration: Contains configuration XML information and other related information
about clocks associated with the organization. See Update Custom Clock Configurations.
Clock Surveys: Contains subfeatures that control how and when surveys appear on
Dayforce clocks in specific scenarios. For example, when configured, clocks might prompt
employees with a survey where they must select a predefined response before the clock
can approve or reject their action, such as time entry. See Clock Survey Configuration. The
following subfeatures are available under Clock Surveys:
Survey Question Setup: Contains the questions and answers that can be associated
with clock surveys in Clock Surveys > Survey Setup. This subfeature contains system-
generated questions and answers that can be associated with clock surveys.
Additionally, this subfeature also supports customized questions and answers.
Survey Setup: This subfeature is used to configure clock surveys and also contains all
available clock surveys.
Clock Survey Groups: Contains options to configure clock survey groups which can be
used to associate surveys with specific clock devices. See Create Clock Survey Groups.

To review culture information for your organization and confirm the default culture:
1. Go to Site Setup > Client and click the Cultures tab.
2. Verify that the list includes all of the cultures that are required by users in your organization.
3. In the Default column, select the checkbox for your organization’s default culture if it’s not
already selected.
4. Click Save.
To add a culture to the list:
1. Go to Site Setup > Client and click the Cultures tab.
2. Click Add.
3. In the Culture column, click the field to expand the drop-down list and select a culture.
